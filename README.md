# Multi-Themed Sports Analytical Dashboard 🏏🏅

An interactive, data-driven web application built using **Streamlit** that provides dual analysis interfaces for two of the world's most celebrated sporting domains: the **Indian Premier League (IPL)** and the **Olympic Games**. 

This dashboard transforms raw historical datasets into structured, actionable insights, offering modular navigation across comprehensive player, team, country, and venue statistics.

---

## 🚀 Key Features

### 🏏 1. IPL Analysis Module (Data: 2008 - 2023)
* **Batsman Analysis:** Deep dive into batting metrics including total runs scored, strike rates, and batting averages to evaluate performance consistency.
* **Bowler Analysis:** Evaluation of bowling trends tracking wickets taken, economy rates, bowling averages, and variations across seasons.
* **Venue Analysis:** Explore how different grounds impact match outcomes through venue-specific statistics, average scores, and pitch condition trends.

### 🏅 2. Olympic Analysis Module
* **Medal Tally:** A dynamic, filterable table showcasing historical Gold, Silver, and Bronze medal standings customizable by year and country.
* **Overall & Country-wise Analysis:** Visualizes dominant countries, track records over time, and sport-specific evolution.
* **Athlete-wise Analysis:** Drill down into individual athlete profiles, tracking multi-edition event participation, age distributions, and milestone performance statistics.

---

## 🛠️ Tech Stack & Libraries
* **Framework:** Streamlit (For building the responsive, interactive UI)
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Plotly / Matplotlib / Seaborn (For intuitive sports metrics mapping)
* **Environment:** Python 3.x
